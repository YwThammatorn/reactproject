# First React Project Webpage

Tools and Languages

Frontend
- CSS, HTML and Javascript
- React Framework
- Axios Library
- Moment Library

Backend
- Node.js and Javascript
- Express.js
- MySQL Database

**Main Requirement**
- User can search product by name, brand or category
- User can view all products and add them to cart
- User can adjust number of products to show per webpage
- User can view selected products in cart, add or delete and order them
- User can adjust number of searchbars to be ordered, save them to localStorage and order them in Order page
- User can view all orders and print receipt for each order individually

Prerequisites
- Database SQL file
- MySQL user profile

Remarks
- This is my archive project when I start learning about React framework for a while
- This project use LocalStorage so it is not the best practice for webpage development
- This project connects with my local MySQL user profile
- All datas in database are virtually created
